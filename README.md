
# Simulador de lanzamiento de dados

Esta aplicación simula el lanzamiento de dados en Python utilizando la librería `dice`.

## Funcionalidad

- Implementa una función que recibe:
  - `amount`: número de lanzamientos
  - `sides`: número de caras del dado
- Devuelve una lista con los resultados obtenidos.
- Ejecuta 5 lanzamientos de un dado de 6 caras.
- Muestra cada resultado con el formato:

  `Lanzamiento {número de lanzamiento} número obtenido {resultado lanzamiento}`

- Incluye una espera de 5 segundos entre lanzamientos.
